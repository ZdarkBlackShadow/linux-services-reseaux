# Configuration terminal de pfsense

![Pfsense logo](./images/pfsense.png)

Pour configurer vos 2 interfaces réseaux, voici ce qu'il vous faudra faire :  

Enter an option : 2 (Set Interfaces IP address )  

Choisissez l'interface réseau que vous souhaitez configurer. Ici je vous donnerais pour exemple l'interface LAN (vtnet1). Il faut donc entrer 2.  

Configure IPv4 address LAN interface via DHCP ? => n  
Enter the new LAN IPv4 address : => 192.168.XX.1 (Vous choisissez, pour notre projet, j'ai pris 192.168.100.1)  
Enter the new LAN IPv4 subnet bit count (1 to 32) : => 24  
For a WAN, enter the new LAN IPv4 upstream gateway address. For a LAN, press ENTER for none : => Laisser vide  
Configure IPv6 address LAN interface via DHCP6 ? => n  
Enter the new LAN IPv6 address. Press ENTER for none : => Laisser vide  
Do you want to enable the DHCP server on LAN ? => y  
Enter the start address of the IPv4 client address range : 192.168.100.100  
Enter the end address of the IPv4 client address range : 192.168.100.200  
Do you want to revert to HTTP as the webConfigurator protocol ? => n (Cela enlèverait le protocole HTTPS pour le passer en HTTP)  

Pfsense vous indiquera alors l'adresse par laquelle accéder à la configuration GUI pour pfsense.  

Vous devriez désormais voir le réseau configuré, comme suis :
